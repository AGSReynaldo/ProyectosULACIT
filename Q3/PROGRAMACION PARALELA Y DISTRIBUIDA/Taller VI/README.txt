En este repositorio de Github se encuentra el taller 6 de la materia de programacion paralela y distribuida, donde se crea el codigo de C y de Java donde ambos envian y reciben mensajes
El codigo C es mas detallado, permitiendo identificar cual es el equipo que envio el mensaje, y tambien permite unicamente enviar un mensaje al ultimo cliente que haya escrito, tambien si se escribe el mensaje "FIN",
Se cierra el servidor UDP de C.
El servidor Java es mas simple, y unicamente envia y recibe mensajes, esto con la intencion de que sea un servicio mas generalizado, y que el cliente de C se sienta mas como un servidor principal o de mayor nivel.
